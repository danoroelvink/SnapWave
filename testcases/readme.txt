Testcases with numbers according to the sections in the paper. 
matlab_utils - matlab functions used by several scripts.

For each test case the following folders:
data - any relevant forcing or validation data
run - could include subfolders for separate runs, each with runsnapwave.bat to run the test and snapwave.inp for the main input. Runsnapwave.bat should refer to the folder with the executable.
output - folder where map and history files are written
scripts - script folder to produce the plots and error statistics
results - folder where the .png figures and statistics are stored.
